# 小红书爆款内容模板库

## 爆款标题公式

### 公式 1: 数字 + 结果 + 情绪
- "5 个 AI 神器让我提前 2 小时下班！真香"
- "3 步配置！AI 自动发飞书日报，太爽了"
- "7 天涨粉 1000+，我做对了这 5 件事"

### 公式 2: 人群 + 痛点 + 解决方案
- "打工人必备！5 个 AI 神器效率翻倍"
- "学生党必看！免费 AI 工具合集"
- "自媒体人救星！这 3 个工具太狠了"

### 公式 3: 对比 + 悬念 + 行动
- "从加班到准点下班，我只用了这 1 个工具"
- "月薪 3 千到 3 万，AI 工具帮我实现了"
- "别再用 XXX 了！这 5 个神器才是真香"

## 爆款封面设计

### 封面文字要求
- **大标题**：10-15 字，占封面 50% 面积
- **数字突出**：用阿拉伯数字，加大加粗
- **对比色**：黄底黑字/白底红字/蓝底白字
- **情绪词**：必备、神器、真香、绝了

### 封面模板
```
┌────────────────────────┐

│                        │

│                        │

│                        │
└────────────────────────┘
```

## 爆款内容结构

### 结构 1: 痛点共鸣型
```
😫 痛点场景（3-4 个具体问题）
   - 让读者觉得"这就是我"

📊 效果数据（真实数据）
   - 使用时间、效率提升、节省时间、收入增长

🛠️ 方案 1-5（详细工具介绍）
   - 每个工具：场景 + 功能 + 步骤 + 效果 + 价格

💡 核心心法（3-5 条经验总结）
   - 让读者觉得"学到了"

✅ 行动清单（可执行步骤）
   - 今天/本周/本月

🎁 福利互动
   - 关注回复获取资料
   - 点赞收藏评论
```

### 结构 2: 对比反差型
```
❌ 使用前：加班到深夜，效率低
✅ 使用后：准点下班，效率翻倍

🤔 我是怎么做到的？

👉 就用了这 5 个 AI 神器

[详细介绍工具]

💬 你在用哪个？评论区聊聊
```

## 爆款文案技巧

### 开头钩子（3 秒抓住注意力）
- "你是不是也..."
- "震惊！原来..."
- "后悔没早知道..."
- "亲测有效！..."

### 中间内容（保持阅读兴趣）
- 多用 emoji（每段 1-2 个）
- 短段落（2-3 行一段）
- 加粗重点（**重点内容**）
- 列表展示（- 要点 1/2/3）

### 结尾行动（引导互动）
- "觉得有用记得点赞 + 收藏"
- "评论区分享你的经验"
- "关注我，获取更多干货"
- "回复 XXX 获取资料"
