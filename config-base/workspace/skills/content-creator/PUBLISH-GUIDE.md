# 内容创作与发布指南

**创建时间**：2026-03-03 22:05

---

## 🎯 功能概述

内容创作助手现在支持**创作 + 发布**一体化流程：

1. ✅ 自动创作内容（报告/文章/演示）
2. ✅ 生成飞书文档
3. ✅ 生成小红书图文素材
4. ✅ 自动发布到指定平台

---

| 平台 | 自动发布 | 素材生成 | 说明 |
|------|----------|----------|------|

| **飞书** | ✅ 支持 | ✅ 支持 | 云文档 |

## 🎨 发布流程

```
🗣️ 用户输入（包含发布指令）
   ↓

   ↓

   ↓

   ↓

   ↓

```

**总耗时**：3-6 分钟  
**自动化程度**：100%

## 📞 故障排除

### 问题 1：发布失败
**现象**：创作成功，但发布失败

**解决**：
1. 检查平台账号登录状态
2. 检查网络环境
3. 查看发布日志
4. 重试或手动发布

### 问题 2：图片渲染失败
**现象**：内容生成成功，但图片卡片失败

**解决**：
1. 检查渲染脚本是否存在
2. 检查 Python 依赖
3. 查看渲染日志
4. 使用默认图片

### 问题 3：标签不相关
**现象**：生成的标签与内容不符

**解决**：
1. 手动指定标签
2. 优化内容提取
3. 调整标签生成算法
