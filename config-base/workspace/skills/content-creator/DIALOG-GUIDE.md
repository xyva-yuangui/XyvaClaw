## 🎨 智能识别规则

### 意图识别
系统会自动识别你的创作意图：

| 关键词 | 识别为 | 默认模板 |
|--------|--------|----------|
| 报告、调研、分析 | create_report | report |
| 文章、博客、公众号 | create_blog | blog |
| 演示、PPT、汇报 | create_presentation | presentation |
| 文档、说明、手册 | create_documentation | documentation |

### 风格识别
系统会自动选择合适的写作风格：

| 关键词 | 识别为 | 适用场景 |
|--------|--------|----------|
| 专业、正式、商业 | professional | 报告/论文/商业文档 |
| 轻松、随意、日常 | casual | 博客/社交媒体 |
| 创意、营销、有趣 | creative | 营销文案/广告 |

### 主题提取
系统会从你的输入中提取创作主题：

**示例**：
- "帮我写一篇关于**AI 漫剧行业分析**的报告"
  → 主题：AI 漫剧行业分析

- "创作一篇**Playwright 浏览器自动化**的技术博客"
  → 主题：Playwright 浏览器自动化

## 📞 故障排除

### 问题 1：识别错误
**现象**：系统识别的意图/风格/模板不对

**解决**：
```
🗣️ 用户：帮我写一篇报告（被识别为博客）

🤖 助手：...

🗣️ 用户：不对，我要的是报告，不是博客

🤖 助手：抱歉，已重新识别为 report 模板，正在重新创作...
```

### 问题 2：主题提取不准确
**现象**：提取的主题太宽泛或太窄

**解决**：
```
🗣️ 用户：帮我写一篇关于"AI 漫剧行业的市场规模和竞争格局分析"的报告

（使用引号明确主题范围）
```

### 问题 3：创作失败
**现象**：系统返回错误

**解决**：
1. 检查网络连接
2. 检查飞书配置
3. 重试或联系管理员
