# 量化选股策略 - 动态权重调整方案 v3.0

## 📊 问题分析

### 当前问题
1. **固定权重** - 估值 25%、成长 25% 等固定不变
2. **无回测验证** - 权重凭经验设定，未经验证
3. **市场状态单一** - 牛市/熊市/震荡市使用相同权重
4. **无自适应能力** - 无法根据市场变化调整

### 解决思路
1. **回测验证** - 用历史数据验证最优权重
2. **动态调整** - 根据市场状态调整权重
3. **自适应学习** - 根据近期表现调整
4. **滚动优化** - 定期重新优化权重

## 🔄 模块 3: 动态权重调整

### 不同市场状态的最优权重

根据历史回测和市场经验，不同市场状态下的最优权重：

#### 牛市权重（进攻型）
| 因子 | 权重 | 理由 |
|------|------|------|
| 估值 | 15% | 牛市估值不重要 |
| 成长 | 30% | 成长股领涨 |
| 动量 | 30% | 强者恒强 |
| 质量 | 10% | 次要 |
| 技术 | 10% | 趋势跟随 |
| 资金 | 5% | 资金推动 |

#### 熊市权重（防御型）
| 因子 | 权重 | 理由 |
|------|------|------|
| 估值 | 35% | 低估值抗跌 |
| 成长 | 15% | 成长不确定 |
| 动量 | 10% | 动量失效 |
| 质量 | 25% | 高质量抗风险 |
| 技术 | 10% | 超卖反弹 |
| 资金 | 5% | 资金流向 |

#### 震荡市权重（平衡型）
| 因子 | 权重 | 理由 |
|------|------|------|
| 估值 | 25% | 均衡配置 |
| 成长 | 25% | 均衡配置 |
| 动量 | 20% | 均衡配置 |
| 质量 | 15% | 均衡配置 |
| 技术 | 10% | 均衡配置 |
| 资金 | 5% | 均衡配置 |

### 动态调整公式
```python
def get_dynamic_weights(market_state, recent_performance):
    # 基础权重（根据市场状态）
    if market_state == "牛市":
        base_weights = {"估值": 0.15, "成长": 0.30, "动量": 0.30, "质量": 0.10, "技术": 0.10, "资金": 0.05}
    elif market_state == "熊市":
        base_weights = {"估值": 0.35, "成长": 0.15, "动量": 0.10, "质量": 0.25, "技术": 0.10, "资金": 0.05}
    else:  # 震荡市
        base_weights = {"估值": 0.25, "成长": 0.25, "动量": 0.20, "质量": 0.15, "技术": 0.10, "资金": 0.05}
    
    # 根据近期表现微调
    # 如果某因子近期表现好，增加权重
    for factor in base_weights:
        recent_return = get_factor_recent_return(factor, days=20)
        if recent_return > 0.05:  # 近期表现好
            base_weights[factor] *= 1.1  # 增加 10%
        elif recent_return < -0.05:  # 近期表现差
            base_weights[factor] *= 0.9  # 减少 10%
    
    # 归一化（确保总和为 1）
    total = sum(base_weights.values())
    weights = {k: v/total for k, v in base_weights.items()}
    
    return weights
```

### 权重自适应调整
```python
def adaptive_adjust(weights, factor_performance):
    """
    根据因子近期表现自适应调整权重
    """
    # 计算每个因子的 IC 值（信息系数）
    ic_scores = {}
    for factor in weights:
        ic = calculate_ic(factor, period="1 月")
        ic_scores[factor] = ic
    
    # IC 值高的因子增加权重
    avg_ic = sum(ic_scores.values()) / len(ic_scores)
    
    for factor in weights:
        if ic_scores[factor] > avg_ic:
            weights[factor] *= 1.1  # IC 高于平均，增加权重
        else:
            weights[factor] *= 0.9  # IC 低于平均，减少权重
    
    # 归一化
    total = sum(weights.values())
    weights = {k: v/total for k, v in weights.items()}
    
    return weights
```

## 📅 模块 5: 滚动优化

### 优化频率
| 优化类型 | 频率 | 内容 |
|---------|------|------|
| **日度** | 每日 | 更新因子表现、微调权重 |
| **周度** | 每周一 | 回顾上周表现、调整市场状态 |
| **月度** | 每月初 | 重新回测、优化权重 |
| **季度** | 每季初 | 全面评估、调整策略参数 |

### 滚动优化流程
```
每月初执行:
  1. 更新历史数据
  2. 重新回测（过去 3 年数据）
  3. 优化权重组合
  4. 验证新权重
  5. 应用新权重
```
