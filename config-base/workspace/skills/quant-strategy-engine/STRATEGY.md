# 量化选股策略 v2.0

## 📊 策略概述

基于市面主流 AI 选股和量化交易方法，结合多因子模型、基本面分析、技术面分析的综合选股策略。

## 🎯 选股因子体系

### 1. 估值因子（Valuation）- 权重 25%
| 因子 | 说明 | 优选范围 | 数据来源 |
|------|------|---------|---------|
| **PE (市盈率)** | 股价/每股收益 | 10-30（合理） | Tushare |
| **PB (市净率)** | 股价/每股净资产 | 1-5（合理） | Tushare |
| **PS (市销率)** | 市值/营业收入 | 1-10（合理） | Tushare |
| **PEG** | PE/净利润增长率 | <1（低估） | Tushare |

### 2. 成长因子（Growth）- 权重 25%
| 因子 | 说明 | 优选范围 | 数据来源 |
|------|------|---------|---------|
| **营收增长率** | 同比增长率 | >20%（高成长） | Tushare |
| **净利润增长率** | 同比增长率 | >20%（高成长） | Tushare |
| **ROE 增长率** | 净资产收益率增长 | >10% | Tushare |
| **毛利率** | 毛利率水平 | >30%（高毛利） | Tushare |

### 3. 动量因子（Momentum）- 权重 20%
| 因子 | 说明 | 优选范围 | 数据来源 |
|------|------|---------|---------|
| **5 日涨幅** | 短期动量 | >0%（正向） | mootdx/Tushare |
| **20 日涨幅** | 中期动量 | >5%（强势） | mootdx/Tushare |
| **60 日涨幅** | 长期动量 | >10%（趋势） | mootdx/Tushare |
| **相对强度 RS** | 相对大盘表现 | >1（跑赢大盘） | 计算 |

### 4. 质量因子（Quality）- 权重 15%
| 因子 | 说明 | 优选范围 | 数据来源 |
|------|------|---------|---------|
| **ROE** | 净资产收益率 | >15%（优秀） | Tushare |
| **ROA** | 总资产收益率 | >5%（良好） | Tushare |
| **资产负债率** | 负债/资产 | <60%（安全） | Tushare |
| **经营现金流** | 现金流状况 | >0（正向） | Tushare |

### 5. 技术因子（Technical）- 权重 10%
| 因子 | 说明 | 优选范围 | 数据来源 |
|------|------|---------|---------|
| **MACD** | 趋势指标 | 金叉（买入信号） | 计算 |
| **RSI** | 超买超卖 | 30-70（正常） | 计算 |
| **成交量比** | 成交量/均量 | >1（放量） | mootdx |
| **布林带** | 波动区间 | 中轨上方（强势） | 计算 |

### 6. 资金因子（Money Flow）- 权重 5%
| 因子 | 说明 | 优选范围 | 数据来源 |
|------|------|---------|---------|
| **北向资金** | 外资流入 | 净流入 | Tushare |
| **主力净流入** | 大单净流入 | 净流入 | Tushare |
| **成交量** | 日均成交 | >1 亿（活跃） | mootdx |
| **换手率** | 成交/流通股本 | 1-5%（适中） | Tushare |

## 📈 评分模型

### 综合评分公式
```
综合评分 = 
  估值因子 × 25% +
  成长因子 × 25% +
  动量因子 × 20% +
  质量因子 × 15% +
  技术因子 × 10% +
  资金因子 × 5%
```

### 因子标准化
```
因子得分 = (因子值 - 最小值) / (最大值 - 最小值) × 100
```

### 选股流程
```
1. 初筛（硬性条件）
   • 市值 > 100 亿
   • 日均成交 > 1 亿
   • 非 ST、非*ST
   • 上市 > 1 年
   
2. 因子计算
   • 计算 6 大类因子
   • 标准化处理
   • 加权综合评分
   
3. 排序筛选
   • 按综合评分排序
   • 选择 Top 10-20
   • 行业分散（单行业 < 30%）
   
4. 风险控制
   • 设置止损位（-5%）
   • 仓位控制（单只 < 20%）
   • 行业分散
```

## ⚠️ 风险提示

1. **历史数据不代表未来** - 回测好不代表实盘好
2. **黑天鹅事件** - 无法预测突发事件
3. **流动性风险** - 小盘股可能无法及时卖出
4. **政策风险** - A 股政策变化频繁
5. **数据延迟** - 实时数据可能有延迟

## 📋 配置参数

```json
{
  "strategy": {
    "name": "多因子量化选股 v2.0",
    "version": "2.0.0",
    "markets": ["A 股"],
    "future_markets": ["港股", "美股"]
  },
  "filters": {

    "min_daily_volume": 1,  // 最小日均成交（亿）

    "min_listing_days": 252 // 最小上市天数（1 年）
  },
  "weights": {
    "valuation": 0.25,
    "growth": 0.25,
    "momentum": 0.20,
    "quality": 0.15,
    "technical": 0.10,
    "money_flow": 0.05
  },
  "risk_control": {
    "stop_loss": -0.05,     // 止损位 -5%
    "max_position": 0.20,   // 单只最大仓位 20%
    "max_industry": 0.30,   // 单行业最大仓位 30%
    "max_stocks": 20        // 最大持股数
  }
}
```
