## Description（简短描述）

xyvaClaw is an open-source AI assistant platform that gets smarter the longer you use it. Built on OpenClaw runtime, it comes with 38+ pre-installed skills (browser automation, video creation, stock analysis, content publishing), a 5-level model fallback system for zero downtime, deep Feishu/Lark integration (112 TypeScript files), and a self-evolution engine that learns from mistakes.

Deploy in one command. Runs locally — your data stays yours.

## First Comment（发布后的第一条评论，建议自己发）

Hi Product Hunt! 👋

I'm 圆规 (yuangui), the solo developer behind xyvaClaw.

The idea started simple: I wanted an AI assistant that actually remembers me and gets better over time. Every tool I tried (ChatGPT, Dify, Coze) either forgot everything each session or required uploading my data to someone else's cloud.

So I built xyvaClaw — it runs on your own machine, has 38 skills ready to go, and genuinely evolves. It records its mistakes, tracks decision outcomes, and proactively discovers tasks to do.

Some highlights:
- **One-liner install**: `bash xyvaclaw-setup.sh --auto` handles everything
- **5-level model fallback**: Never goes down, even when DeepSeek has an outage
- **Deep Feishu integration**: 112 TypeScript files covering nearly every Feishu API
- **Self-evolution**: The longer you use it, the smarter it gets

It's MIT licensed, free for personal and commercial use.
