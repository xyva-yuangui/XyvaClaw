## xyvaClaw 是什么？

一句话：**一个会记忆、会学习、会主动干活的 AI 助手平台。**

它不是又一个套壳 ChatGPT。它是一个跑在你自己机器上的 AI 系统，有 38 个内置技能，能操作浏览器、写文档、做视频、管理飞书、量化选股，还能**从错误中学习，越用越聪明**。

核心差异：

| 普通 AI 聊天 | xyvaClaw |
|---|---|
| 每次对话从零开始 | 四层记忆系统，永远记得你 |
| 只能聊天 | 38 个技能，能直接帮你干活 |
| 单一模型 | 五级模型容灾，永不掉线 |
| 手动配置 | 一条命令部署，两分钟上线 |
| 被动回答 | 主动发现任务并执行 |
| 聊越久越容易"忘事" | 无损上下文引擎，长对话不丢信息 |

## 自我进化是怎么回事？

这是我最花心思的部分。

大多数 AI 工具是**静态的**——你用了一年，它还是第一天的样子。xyvaClaw 不一样：

1. **错误学习**：犯过的错自动记录到 `.learnings/ERRORS.md`，下次绝不重复
2. **效果追踪**：每个决策都追踪结果，好的策略被强化，差的被淘汰
3. **主动反思**：周期性分析自己的行为模式，主动优化
4. **主动行动**：不等你问，它会读你的待办事项、扫描日志、发现问题并处理

用了三个月后，我的 xyvaClaw 已经学会了：
- 我喜欢的代码风格
- 我常用的分析框架
- 我的飞书群发消息习惯
- 哪些股票筛选策略对我有效

**越用越懂你，这不是 slogan，是真的。**

## 技术架构

xyvaClaw 基于 [OpenClaw](https://github.com/nicepkg/openclaw) 运行时深度增强。你可以理解为 OpenClaw 的**增强版**——相同的引擎，但我加了大量实战打磨的东西：

- **五级模型容灾**：DeepSeek → Qwen → Kimi → DeepSeek Reasoner → Qwen Max，任何一个挂了自动切到下一个
- **无损上下文引擎（Lossless-Claw）**：聊到 10 万 token 也不会"忘事"
- **飞书深度集成**：112 个 TypeScript 源文件，覆盖飞书几乎所有 API
- **38 个预装技能**：浏览器自动化、文档处理、视频制作、量化选股、小红书发布...

## 安装有多简单？

```bash
# macOS / Linux 一条命令
DEEPSEEK_API_KEY=sk-你的密钥 \
  bash -c 'git clone https://github.com/xyva-yuangui/XyvaClaw.git && cd XyvaClaw && bash xyvaclaw-setup.sh --auto'
```

没错，就一行。`--auto` 模式会自动处理一切：
- 没有 Node.js？自动装
- 没有 Python？自动装
- 没有 ffmpeg？自动装
- API Key？从环境变量自动注入
- 开机自启？自动配置

如果你喜欢图形界面，去掉 `--auto` 就会弹出 Web 配置向导，在浏览器里点点就行。

## 38 个技能都能干什么？

挑几个有意思的说：

- **browser-pilot**：自动打开浏览器帮你操作网页，填表、截图、爬数据
- **auto-video-creator**：AI 生成视频，配音+字幕+剪辑一条龙
- **quant-strategy-engine**：A 股多因子量化选股，每天自动推送
- **xhs-publisher**：小红书内容自动生成+一键发布
- **rag-knowledge-base**：本地知识库，喂文档进去就能问答
- **content-creator**：多平台内容生成，一套内容适配多个平台

完整技能列表见 GitHub 仓库。

## 和其他 AI 工具的区别

有人会问：这和 Dify / Coze / FastGPT 有什么区别？

最大的区别：**xyvaClaw 跑在你自己的机器上，数据完全私有。**

它不是一个 SaaS 平台，不需要你把数据上传到别人的服务器。你的记忆、你的对话、你的文件，全在 `~/.xyvaclaw/` 里，只有你能看到。

其次，它是**主动的**。Dify/Coze 本质上还是"你问我答"的工作流工具。xyvaClaw 有自己的心跳机制、定时任务、主动反思——它真的在"活着"。
