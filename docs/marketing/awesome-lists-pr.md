## 1. awesome-ai-agents

> 仓库：https://github.com/e2b-dev/awesome-ai-agents
> 操作：Fork → 编辑 README.md → 在合适分类下添加 → 提交 PR

### 添加内容

```markdown
- [xyvaClaw](https://github.com/xyva-yuangui/XyvaClaw) - Self-evolving AI assistant platform with 38+ skills, 5-level model fallback, deep Feishu/Lark integration, and one-click deployment. Runs locally. MIT licensed.
```

### PR 标题

Add xyvaClaw - Self-evolving AI assistant platform

### PR 描述

Adding xyvaClaw, an open-source AI assistant platform built on OpenClaw runtime.

**Key features:**
- 38+ pre-installed skills (browser automation, video creation, stock analysis, etc.)
- Self-evolution engine (error learning, effect tracking, proactive reflection)
- 5-level model fallback for zero downtime
- Deep Feishu/Lark integration (112 TypeScript files)
- 4-tier memory system (session → daily → long-term → knowledge graph)
- One-command deployment with `--auto` unattended mode
- Runs locally, data stays private

Website: https://www.xyva.fun
License: MIT
